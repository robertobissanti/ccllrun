Nella directory corrente crea il file solution.py. Implementa la funzione `merge_intervals(intervals)` che riceve una lista di coppie di interi, non modifica l'input, normalizza ogni coppia invertita, unisce intervalli sovrapposti o adiacenti e restituisce una lista di tuple ordinata. Esempio: `[(5,1),(2,4),(10,10),(11,12)]` deve produrre `[(1,5),(10,12)]`. Gestisci lista vuota, duplicati e valori negativi. Usa solo la libreria standard. Non creare altri file. Alla fine rispondi con una spiegazione di massimo 60 parole.
