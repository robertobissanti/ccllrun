La funzione normalizza ogni coppia con `sorted()` (garantisce `(min, max)`), rimuove i duplicati mantenendo l'ordine, ordina per inizio, poi fonde gli intervalli sovrapposti o adiacenti (`start <= ultimo_end + 1`) aggiornando l'estremo destro al massimo. Restituisce una lista di tuple ordinata. Complessità: O(n log n) per il sort, O(n) per la fusione. Gestisce correttamente lista vuota, negativi e duplicati.
